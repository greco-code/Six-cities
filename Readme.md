### About:

- React App
- Redux
- Fetch.API
- Leaflet.js
- Webpack

### Pages:

- Main
- Info-page for every hotel
- Favorites page
- Login page

### You can:

- Choose a city
- Login/logout
- Add hotel to favorites
- Remove hotel from favorites
- Write a comment

### Commands

At first, change directory to /project

`npm start`
Runs dev-server ([http://localhost:8080](http://localhost:8080) by default)

`npm run build`
Builds project
